# **App Name**: DukaanXP

## Core Features:

- Offline Data Storage: All business data (customer, supplier, stock, transactions) is stored locally on the device for 100% offline access.
- Real-Time Auto-Save: All data input into forms is saved instantly, preventing data loss and ensuring seamless operation.
- Smart Search & Filtering: Enables quick lookups of customers, products, and transactions by various criteria such as name, date, and keywords.
- Bilingual UI Toggle: Supports both Urdu and English languages with a quick toggle option to switch between languages.
- Automated Alerts & Reminders: Configurable alerts and reminders for pending payments, low stock levels, and employee salary dates.
- Printable & Exportable Reports: Generates and exports detailed business reports in PDF and Excel formats for further analysis and sharing.

## Style Guidelines:

- Primary color: Blue (#1E88E5) for trust and professionalism.
- Accent color: Amber (#FFC107) for highlighting key actions and information.
- Background color: Light gray (#F5F6FA) to ensure readability and reduce eye strain.
- English font: 'Poppins' (sans-serif) for a clean and modern look.
- Urdu font: 'Jameel Noori Nastaleeq' for authentic representation and readability. Note: currently only Google Fonts are supported.
- Card-based layout with rounded corners (15-20px radius) for a modern, user-friendly interface.
- Intuitive and minimalist icons to represent different modules and actions.
- Subtle transitions and animations to enhance user experience and provide visual feedback.
